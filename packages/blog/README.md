Monorepo 의 Blog 패키지 입니다. 학습하고 경험한 것들을 남기는 블로그 입니다. 현재는 static 데이터로 이루어져 있습니다. 메인 페이지와 블로그 목록 페이지 , post 페이지로 이루어져 있습니다.

1. **노드 버전 (>= 20.0.0)**

   ```sh
   node --version
   ```

2. **yarn 버전 (4.4.1)**
   ```sh
   yarn --version
   ```
